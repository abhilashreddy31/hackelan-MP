# hackelan-MP
HackÉlan is a hybrid AI-driven framework proposed to address the fragmented, unstructured, and institution-restricted nature of student engagement in technical and cultural events across higher education ecosystems. Existing event discovery mechanisms lack centralized multi-campus collaboration, credibility-based ranking, real-time personalized recommendations, and structured performance analytics. To overcome these limitations, HackÉlan introduces a collaborative filtering–based event recommendation engine and a hybrid ranking model that integrates weighted participation metrics with predictive performance scoring. The platform enables institutions to publish, monitor, and analyze events through unified dashboards, while students benefit from personalized event suggestions, cross-campus collaboration opportunities, milestone recognition, and transparent performance evaluation. The proposed architecture integrates event aggregation, interaction analytics, predictive modeling, and fairness-aware ranking within a scalable microservices-based infrastructure. Experimental evaluation on a large-scale simulated dataset demonstrates improvements in recommendation precision, ranking stability, and participation diversity compared to traditional decentralized systems. HackÉlan establishes a scalable foundation for national-level academic event ecosystems and provides a structured pathway toward data-driven skill development and institutional benchmarking in higher education.
